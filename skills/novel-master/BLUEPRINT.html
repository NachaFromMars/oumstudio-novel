📦 6 Mô-đun Chính

Mỗi mô-đun giải quyết một mối nguy nhất quán — cùng chia sẻ tầng dữ liệu chung.

LÕI
📚

Quản Lý Kinh Điển

Tạo, sửa, xoá Kinh Điển Nhân Vật + Kinh Điển Thế Giới. Nguồn sự thật duy nhất cho mọi thực thể trong tiểu thuyết.

  • Hồ sơ nhân vật (thuộc tính, mối quan hệ, dòng thời gian)
  • Thực thể thế giới (địa danh, phe phái, vật phẩm, hệ thống phép thuật)
  • Tự động nhập từ các chương đã viết
  • Tìm kiếm + truy vấn + xuất để nạp ngữ cảnh
QUÉT
🔍

Kiểm Tra Tính Liền Mạch

30+ quy tắc quét mâu thuẫn — dòng thời gian, nhân vật, thế giới, mạch truyện. Chấm điểm độ tin cậy tránh báo nhầm.

  • Nhất quán thời gian (sáng→chiều→tối, thời gian di chuyển)
  • Phát hiện nhân vật "dịch chuyển tức thời"
  • Nhảy cấp lực lượng bất thường
  • Báo cáo: NGHIÊM TRỌNG / CẢNH BÁO / GHI CHÚ + gợi ý sửa
PHÂN TÍCH
✍️

Phân Tích Văn Phong

Phân tích văn phong tiếng Việt — phát hiện lệch giọng giữa các cung, giữ giọng văn nhất quán xuyên suốt mọi chương.

  • Thống kê độ dài câu / đoạn
  • Theo dõi tỷ lệ đối thoại / tự sự
  • Chỉ số vốn từ + mức lặp từ
  • Phát hiện lệch giọng so với đường cơ sở (ngưỡng ±20%)
THEO DÕI
📈

Theo Dõi Nhịp Truyện

Phân loại nhịp đoạn 5 cấp, vẽ đường cong căng thẳng, phát hiện lặp nhịp — tránh mệt mỏi cho người đọc.

  • 5 cấp: TĨNH → DÂNG → CĂNG → CAO TRÀO → HẠ NHIỆT
  • Biểu đồ nhịp dạng ký tự ASCII
  • Phát hiện đoạn đơn điệu liên tiếp
  • Cảnh báo thiếu hạ nhiệt sau cao trào
GIỌNG
🎭

Quản Lý Giọng Nhân Vật

Mỗi nhân vật một giọng riêng — cách nói, khẩu ngữ, vốn từ. Không ai giống ai.

  • Tự động trích xuất đối thoại theo nhân vật
  • Hồ sơ giọng: đại từ, vốn từ, hài hước, kiểu câu
  • Phát hiện giọng nhân vật trùng nhau
  • Kiểm tra nhất quán giọng theo chương

🔨 Lộ Trình Xây Dựng — 9 Giai Đoạn, 20 Bước

Quy trình StepForge: mỗi bước 4.000–6.000 token, đơn lập, thẩm định, cổng chặn cứng, tuần tự.

GĐ1 — NỀN TẢNG DỮ LIỆU

Bước 1–2: Thiết Kế Lược Đồ

Bước 1: Lược đồ JSON cho Kinh Điển Nhân Vật + Thế Giới, quy tắc phiên bản, di chuyển dữ liệu
Bước 2: Mô hình nhịp truyện 5 cấp + mẫu hồ sơ giọng nhân vật
GĐ2 — QUẢN LÝ KINH ĐIỂN

Bước 3–5: Tạo-Sửa-Xoá + Truy Vấn + Nhập Liệu

Bước 3: Tiện ích chung + Thao tác cơ bản (tạo/cập nhật/lấy/liệt kê/xoá)
Bước 4: Truy vấn + tìm kiếm + dòng thời gian + mối quan hệ + xuất dữ liệu
Bước 5: Nhập từ các chương (tự động trích xuất thực thể)
GĐ3 — KIỂM TRA LIỀN MẠCH

Bước 6–8: Quy Tắc + Bộ Quét + Báo Cáo

Bước 6: 30+ quy tắc kiểm tra (thời gian, nhân vật, thế giới, mạch truyện)
Bước 7: Bộ quét xử lý từng đoạn + chấm điểm độ tin cậy
Bước 8: Tạo báo cáo (NGHIÊM TRỌNG / CẢNH BÁO / GHI CHÚ + gợi ý)
GĐ4 — PHÂN TÍCH VĂN PHONG

Bước 9–10: Thống Kê + Phát Hiện Lệch Giọng

Bước 9: Số liệu văn phong tiếng Việt (câu, đối thoại, vốn từ, ngôn ngữ đời thường)
Bước 10: Phát hiện lệch giọng — so sánh các chương với đường cơ sở
GĐ5 — THEO DÕI NHỊP TRUYỆN

Bước 11–12: Phân Loại + Đường Cong + Nhận Dạng Khuôn Mẫu

Bước 11: Phân loại nhịp đoạn (từ khoá + cấu trúc → 5 cấp)
Bước 12: Tạo đường cong căng thẳng + phát hiện khuôn mẫu (đơn điệu, cung phẳng)
GĐ6 — QUẢN LÝ GIỌNG NHÂN VẬT

Bước 13–14: Xây Hồ Sơ + Kiểm Tra Nhất Quán

Bước 13: Xây hồ sơ giọng (trích đối thoại → phân tích khuôn mẫu lời nói)
Bước 14: Kiểm tra nhất quán giọng + ma trận tương đồng giữa nhân vật
GĐ7 — GIAO DIỆN DÒNG LỆNH + TÀI LIỆU

Bước 15–16: Trình Bọc CLI + SKILL.md

Bước 15: CLI Commander.js — 5 lệnh (kinhdien/kiemtra/phantich/nhip/giong)
Bước 16: Hoàn thiện SKILL.md + tài liệu tham chiếu + hướng dẫn tích hợp
GĐ8 — XUẤT BẢN (v1.2 MỚI)

Bước 17–19: Gộp Chương + Đa Định Dạng + Audiobook

Bước 17: Combine — gộp N chương → 1 file .md có mục lục tự động
Bước 18: Xuất đa định dạng — EPUB (pandoc) + PDF (wkhtmltopdf, A5) + HTML reader (responsive, dark mode) + QR code
Bước 19: Audiobook — edge-tts chunk processing + ffmpeg concat → mp3 mỗi chương
GĐ9 — BẢN THIẾT KẾ + ĐÓNG GÓI

Bước 20: Bản Thiết Kế HTML + Package Cài Đặt

Bước 20: Trang HTML bản thiết kế cao cấp + README.md + đóng gói .zip hoàn chỉnh ✅

⌨️ Tra Cứu Dòng Lệnh

5 lệnh, 1 điểm vào duy nhất: node scripts/guardian.mjs

Lệnh Lệnh con Mô tả
bible create <loại> <tên> Tạo thực thể (nhân vật / địa danh / phe phái / vật phẩm)
get <mã> Lấy thông tin thực thể
list [--type] Liệt kê tất cả thực thể
search <từ khoá> Tìm kiếm toàn bộ Kinh Điển
import <thư mục> Nhập thực thể từ các chương
export [--format] Xuất bản tóm tắt dễ đọc
sync <thư mục> Đồng bộ hoàn toàn Kinh Điển từ các chương
check <thư mục> Quét toàn bộ các chương
<tệp> Quét 1 chương
--range 21-40 Quét phạm vi chương
analyze <thư mục> Phân tích toàn bộ văn phong
--baseline 1-20 Đặt đường cơ sở từ các chương mẫu
--compare 21-40 So sánh với đường cơ sở
pace <thư mục> Phân tích nhịp truyện toàn bộ
--visual Biểu đồ đường cong căng thẳng dạng ký tự
--chapter N Nhịp truyện 1 chương
voice build <nhân vật> <thư mục> Xây hồ sơ giọng nhân vật
check <thư mục> Kiểm tra nhất quán giọng
compare <nv1> <nv2> So sánh giọng hai nhân vật
publish combine <thư mục> Gộp N chương → 1 file .md có mục lục
epub <đầu vào> Tạo .epub chuẩn
pdf <đầu vào> Xuất PDF đẹp (A5, typography)
html <đầu vào> Tạo web reader (responsive + dark mode)
qr <nội dung> Tạo QR code chia sẻ
audiobook <thư mục> Chuyển chương → mp3 audiobook
all <thư mục> Pipeline tổng: gộp → epub+pdf+html+qr
